Number Pair Match

Game Instructions

Understanding Difficulty Levels

  • Easy: Incorrect cards stay face-up for 1.5 seconds. Correct matches earn 10 points.
  • Medium: Incorrect cards stay face-up for 1.0 second. Correct matches earn 15 points.
  • Hard: Incorrect cards flip back fast (0.5 seconds). Correct matches earn 25 points!

How the Game Works

In this game, the player flips two cards at a time and tries to find a valid number pair.

A valid pair depends on the selected pair type:

  • Number and Word: for example 5 and five
  • Sum and Answer: for example 3 + 4 and 7
  • Mixed: a combination of both types

Steps to Play

  1. Enter your player name on the launcher page.
  2. Select board size and difficulty.
  3. Select the pair type.
  4. Choose optional settings such as timer and hints.
  5. Open the game window.
  6. Click Start Game.
  7. Flip two cards to try to find a correct pair.
  8. Continue until all pairs are matched.

Game Rules & Difficulty

Match pairs of numbers with their corresponding words or equations.

  • Easy: 1.5s preview on miss | 10 pts per match.
  • Medium: 1.0s preview on miss | 15 pts per match.
  • Hard: 0.5s preview on miss | 25 pts per match.

Note: You have 3 lives. Every incorrect match subtracts a life. If you reach 0, the game ends.

Buttons

  • Start Game starts a new game
  • Show Hint highlights one possible card
  • Save Session stores the current game state
  • Load Session restores a saved game
  • Reset Game clears the current game
  • Back to Settings returns to the launcher page
